## About the role

You'll build web applications using SvelteKit on the frontend and Go on the backend. Projects like [microblog](https://github.com/spikenardco/microblog) are the kind of work you'll do — taking well-understood problems and implementing them with modern tools, cleanly and completely.

## What you'll do

- Design and build SvelteKit frontends with TypeScript
- Write Go backend services with chi, sqlx, and SQLite
- Own features from database schema to deployed UI
- Ship in small, working increments — not month-long feature branches
- Write documentation that helps the next person (including future you)

## What we're looking for

- Proficiency with a modern frontend framework — Svelte preferred, but React/Vue experience translates
- Backend experience with Go, or a strong willingness to learn it
- Comfort writing SQL directly, not just through an ORM
- You default to the simplest solution and add complexity only when forced to
- You've shipped something to production and maintained it afterward

## Nice to have

- Experience with SvelteKit specifically (form actions, load functions, adapters)
- Familiarity with SQLite and its strengths for small-to-medium applications
- A side project or open-source contribution you're proud of
